## ⚡ Features That Go BRRRR

- **🔥 Lightning Fast Search:** Powered by React Hooks (`useState`) and the OMDB API. Instant results as soon as you smash *Enter*!
- **🖤 Red & Black Aesthetic:** A meticulously crafted dark theme (`linear-gradient(135deg, #1a0505, #000000)`) with aggressive neon crimson highlights to make your eyes bleed with joy! ❤️‍🔥
- **✨ Glassmorphism Magic:** Why use boring solid modals when you can have a futuristic `backdrop-filter: blur(12px)` glass-like floating popup? Feels like browsing from a spaceship! 🛸
- **🦾 Bulletproof:** Zero crashes. Missing poster? Boom—you get a slick fallback! Missing plot? Boom—custom fallback text! It's *unstoppable*.
- **🎬 Micro-Animations:** Because a static UI is a dead UI. Enjoy juicy `scale(1.05)` hover snaps and buttery `fadeIn` sequences!

## 🚀 How to Launch This Beast

If you want to run the Cineverse instance locally on your battle station, follow these sacred steps:

1. **Clone it.** `git clone ...` (You know the drill).
2. **Install the juice.** Run `npm install` in the terminal to grab all dependencies.
3. **Ignition.** Run `npm start`. If you're on a newer Node.js (17+), it'll handle the OpenSSL legacy provider automatically thanks to our battle-tested `package.json` setup!
4. **Witness Greatness.** Open your browser to `http://localhost:3000` and start searching for movies like "The Matrix" or "Batman" or "Morbius"!

## 🛠️ Tech Stack!

- **Frontend Core:** React.js (Hooks, Functional Components)
- **Styling:** Pure, Vanilla CSS. No tailwind bloat here! Hand-crafted gradients and shadows.
- **Data Hookup:** Axios bridging us directly into the OMDB API Matrix.
